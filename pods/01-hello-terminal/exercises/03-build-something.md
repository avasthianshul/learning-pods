# Exercise 3: Build Something

**Time:** 10 minutes

## Steps

1. In your Claude Code session (inside `my-first-project/`), type this prompt:

   > Create an HTML page that says Hello [YOUR NAME]. Use a dark background (#1a1a2e) and white text. Add today's date and a fun fact about sustainability.

2. Claude will create a file (probably `index.html`). It will tell you the filename.

3. Open the file in your browser:
   - **Windows:** In File Explorer, navigate to `C:\Users\yourname\claude_projects\my-first-project\` and double-click `index.html`
   - **Mac:** In Finder, navigate to the same folder and double-click
   - Or ask Claude: "How do I open this file in my browser?"

4. You should see your webpage! Now iterate — ask Claude:

   > Change the background to green and add a section about what I do at Oren

5. Refresh your browser — see the changes instantly.

6. Try one more change:

   > Add a photo placeholder and make the layout look professional with some padding

## What Just Happened

You described what you wanted in plain English. Claude wrote the code. You previewed it in your browser. You asked for changes. Claude made them. **This is the feedback loop you'll use for everything.**

## Success Criteria

- [ ] You have an HTML page with your name on it
- [ ] You've modified it at least twice by talking to Claude
- [ ] You can see it in your browser
