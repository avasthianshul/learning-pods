# Exercise 4: Clone and Explore

**Time:** 5 minutes

## What You'll Do

Download the entire course materials from GitHub to your computer, then explore the Pod 01 revision app in your browser.

## Steps

### Part 1: Clone the Repo

1. In your Claude Code session, type:

   > Clone the learning pods repository from GitHub to my claude_projects folder. The repo URL is https://github.com/avasthianshul/learning-pods.git

2. Claude will download the repository. When it's done, you'll have a new folder at `~/claude_projects/learning_pods/`

3. Tell Claude:

   > Open the learning_pods folder

### Part 2: Explore the Lecture Notes App

1. Open this file in your browser:
   - **Windows:** Navigate to `C:\Users\yourname\claude_projects\learning_pods\pods\01-hello-terminal\app\` and double-click `index.html`
   - **Mac:** Navigate to the same path in Finder and double-click
   - Or ask Claude: "How do I open the file pods/01-hello-terminal/app/index.html in my browser?"

2. You'll see the interactive lecture notes for today's session

3. Try these things:
   - **Click any command** — it copies to your clipboard
   - **Browse each section** — Terminal Basics, Folder Map, Claude Code, GitHub, Clone & Explore
   - **Use the navigation links** at the top to jump between sections

4. **Bookmark this page** — it's your go-to reference when you forget a command

## Success Criteria

- [ ] learning_pods repo is cloned to `~/claude_projects/learning_pods/`
- [ ] You can see the repo files (try `ls` in the terminal)
- [ ] You've opened Pod 01's `index.html` in your browser
- [ ] You've clicked at least one command to copy it
