# Exercise 2: Your First Project

**Time:** 5 minutes

## Steps

1. Make sure you're in your projects folder:
   ```
   cd ~/claude_projects
   ```

2. Create a new project folder:
   ```
   mkdir my-first-project
   ```

3. Go into it:
   ```
   cd my-first-project
   ```

4. Start Claude Code:
   ```
   claude
   ```

5. Initialize the project:
   ```
   /init
   ```

6. Claude will ask you a few questions about your project. Answer them in plain English — there are no wrong answers.

7. When it's done, look at the CLAUDE.md file it created. This is your project's rulebook — Claude reads it every time it starts working here.

## Success Criteria

- [ ] You have a `my-first-project/` folder inside `~/claude_projects/`
- [ ] There's a `CLAUDE.md` file in it
- [ ] You understand that CLAUDE.md = instructions for Claude
