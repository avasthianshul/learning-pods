# Exercise 2: Your First Project

**Time:** 10 minutes

## Steps

1. Make sure you're in your projects folder:
   ```
   cd ~/claude_projects
   ```

2. Create a project folder and go into it:
   ```
   mkdir my-first-project
   cd my-first-project
   ```

3. Start Claude Code:
   ```
   claude
   ```

4. Type this prompt (replace [YOUR NAME] with your actual name):

   > Create an HTML page that says Hello [YOUR NAME]. Use a dark background (#1a1a2e) and white text. Add today's date and a fun fact about sustainability.

5. Claude will create a file (probably `index.html`). Open it in your browser:
   - **Windows:** In File Explorer, navigate to `C:\Users\yourname\claude_projects\my-first-project\` and double-click `index.html`
   - **Mac:** In Finder, navigate to the same folder and double-click
   - Or ask Claude: "How do I open this file in my browser?"

6. You should see your webpage! Now iterate — tell Claude:

   > Change the background to a gradient and add a section about what I do at Oren. Make it look professional.

7. Refresh your browser — see the changes instantly.

8. Try one more:

   > Add a photo placeholder and make the layout look like a professional profile page

## What Just Happened

You described what you wanted in plain English. Claude wrote the code. You previewed it in your browser. You asked for changes. Claude made them. **This describe-and-build loop is how you'll work from now on.**

## Success Criteria

- [ ] You have an HTML page with your name on it
- [ ] You've modified it at least twice by talking to Claude
- [ ] You can see it in your browser
- [ ] You used Ctrl+C at least once (try it — it's your emergency brake)
