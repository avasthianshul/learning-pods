# Exercise 5: Essential Commands

**Time:** 5 minutes

## Steps

1. Ask Claude to do something that takes a moment:

   > Create a Python script that prints the numbers 1 to 100, one per line, with a half-second delay between each

2. While it's running, press **Ctrl+C** to stop it. Nothing breaks — you just stopped the current action.

3. Type `/resume` — Claude picks up right where you left off, remembering your whole conversation.

4. Type `/context` — see what files and information Claude is currently working with.

5. Type `/memory` — see what Claude has remembered about your project and preferences.

## Bonus: Two Tabs

1. Open a second Warp tab (Ctrl+T or Cmd+T)
2. Navigate to the same project: `cd ~/claude_projects/my-first-project`
3. Start a new Claude session: `claude`
4. Give it a different task: "Create a second page called about.html with information about Oren — what the company does and why sustainability matters"
5. Switch between tabs — you now have **two builders working for you**

Start with 2 tabs maximum. Once you're genuinely comfortable, you can try 3. But 2 is the sweet spot for most people.

## Success Criteria

- [ ] You've used Ctrl+C to stop a running task
- [ ] You've used /resume to continue a session
- [ ] You've looked at /context and /memory
- [ ] (Bonus) You've run Claude in two tabs simultaneously
