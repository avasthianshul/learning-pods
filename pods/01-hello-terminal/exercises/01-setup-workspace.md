# Exercise 1: Set Up Your Workspace

**Time:** 5 minutes

## Steps

1. Open Warp terminal

2. Create your projects home base:
   ```
   mkdir ~/claude_projects
   ```

3. Go into it:
   ```
   cd ~/claude_projects
   ```

4. Verify you're in the right place:
   ```
   pwd
   ```
   You should see something like `/c/Users/yourname/claude_projects` (Windows) or `/Users/yourname/claude_projects` (Mac)

5. Now open File Explorer (Windows) or Finder (Mac) and navigate to the same folder. You should see it there — **same folder, two views.**

## Success Criteria

- [ ] `~/claude_projects/` folder exists
- [ ] You can see it in both the terminal AND your file manager
- [ ] You know what `pwd` shows you
