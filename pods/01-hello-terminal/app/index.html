Terminal Phrasebook

What you want to do → what to type. Click any command to copy it.

Getting Around

See what's in this folder
ls
Like opening a folder in File Explorer
Go into a folder
cd folder-name
Like double-clicking a folder
Go back / up one folder
cd ..
Like clicking the back arrow
Where am I right now?
pwd
Like reading the address bar
Go to my home folder
cd ~
Like clicking "Home" in the sidebar

Creating Things

Create a new folder
mkdir folder-name
Like right-click → New Folder
Create your projects home base
mkdir ~/claude_projects
Your home base for all Claude work

Claude Code

Start Claude Code
claude
Opens your AI builder
Set up a new project
/init
Creates CLAUDE.md — your project's rulebook
Pick up where I left off
/resume
Continues your last conversation
What does Claude see right now?
/context
Shows Claude's current view
What has Claude remembered?
/memory
Shows persistent memories
STOP what Claude is doing
Ctrl+C
Emergency brake — won't break anything
