Terminal Basics — 5 Commands

See what's in this folder
ls
list — lists the contents
Like opening a folder in File Explorer
Go into a folder
cd folder-name
change directory
Like double-clicking a folder
Go back / up one folder
cd ..
change directory to parent (.. = one level up)
Like clicking the back arrow
Where am I right now?
pwd
print working directory
Like reading the address bar
Create a new folder
mkdir folder-name
make directory
Like right-click → New Folder
Go to my home folder
cd ~
~ is shorthand for your home folder
Like clicking "Home" in the sidebar

Claude Code — Your AI Builder

Start Claude Code
claude
Opens your AI builder in the current folder
STOP what Claude is doing
Ctrl+C
Emergency brake — won't break anything, use freely
Example first prompt
"Create an HTML page that says Hello World with a dark background and today's date"
Describe what you want in plain English — Claude builds it

Why Git & GitHub?

Google Drive = documents, presentations, resources. GitHub = dashboards, tools, automations — anything that evolves.

Remote
Your project on GitHub (online copy)
Local
Your project on your computer (downloaded copy)
Push
Upload your changes to GitHub
Pull
Download latest changes from GitHub

GitHub Setup — 3 Steps

Once connected, your work is saved, shareable, and version-tracked.

1
Ask Claude to install GitHub CLI
"Install the GitHub command-line tool for me"
Type this inside Claude Code — Claude handles Mac vs Windows for you
2
Log in to GitHub (in Warp, not Claude)
gh auth login
Open a new Warp tab → type this → follow browser prompts → come back to Claude
3
Verify it worked
"Check if GitHub is connected"
Back in Claude Code — Claude runs gh auth status to confirm
