# Oren Learning Pods: Claude Code Training Series

A 13-week hands-on training series to take the Oren team from terminal beginners to full-stack confident Claude Code users.

## Quick Start

1. **Install Warp terminal** — [warp.dev](https://www.warp.dev/) (Windows + Mac)
2. **Clone this repo:**
   ```bash
   git clone <repo-url> ~/claude_projects/learning_pods
   cd ~/claude_projects/learning_pods
   ```
3. **Run setup** (Week 2 — we'll do this together):
   - Windows: `powershell setup/install-windows.ps1`
   - Mac: `bash setup/install-mac.sh`
4. **Verify everything works:** `python setup/verify.py`

## How Each Session Works

**Before the session (15-20 min):**
- Read the pod's `README.md` and flip through `README.pptx`
- Follow any pre-work instructions

**During the session (60 min):**
- 30 min: Live walkthrough of the presentation
- 30 min: Hands-on exercises (everyone works together, ask questions freely)

**After the session:**
- Practice with the exercises at your own pace
- Use the pod's browser app (`app/index.html`) as a reference

## What You Need

- A computer (Windows or Mac)
- [Warp terminal](https://www.warp.dev/) installed
- A Claude Pro or Max subscription
- A GitHub account (free)
- Willingness to type things into a terminal (we promise it's not scary)

## Having Trouble?

1. Open Warp terminal
2. Type `claude`
3. Tell Claude what's wrong — it will help you fix it
4. If Claude can't fix it, message #learning-pods on Slack
